Lista de utilidades Tailwind
- `flex` / `flex-col` / `flex-row` — Acomodar cosas en fila o columna
- `grid` / `grid-cols-2` / `grid-cols-4` — Hacer la cuadrícula de tarjetas
- `container` / `mx-auto` — Centrar el contenido en la página
- `px-4` / `py-8` / `gap-8` — Espaciado (padding y separación)
- `lg:flex-row` / `sm:grid-cols-2` — Cambiar layout según el tamaño de pantalla
- `hidden` / `md:flex` — Ocultar o mostrar cosas en móvil o desktop o tablet, responsivo
